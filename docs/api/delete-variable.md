Makes sure that [@ng-apimock/core](https://github.com/ng-apimock/core) will delete the variable.

## Usage
```typescript
deleteVariable(key);
```
 
## Parameters
| Name | Type | Details |
| ---- | ---- | ------- |
| <code><var>key</var></code> | string | the key |
 
## Example 
```typescript
deleteVariable('now');
```
Executing this will delete the variable with key `now`.
